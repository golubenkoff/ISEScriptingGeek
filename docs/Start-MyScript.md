# Start-MyScript

## SYNOPSIS

Launch the current script file

## SYNTAX

```yaml
Start-MyScript [[-Path] <String>]
```

## DESCRIPTION

Use this command to launch the current script file with Show-Command. Fill in the form to run the script.
