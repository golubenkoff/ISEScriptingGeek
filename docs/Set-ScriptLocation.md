# Set-ScriptLocation

## SYNOPSIS

Set location to current script folder.

## SYNTAX

```yaml
Set-ScriptLocation [<CommonParameters>]
```

## DESCRIPTION

Use this command to change location to match that of the current script in the PowerShell ISE.
