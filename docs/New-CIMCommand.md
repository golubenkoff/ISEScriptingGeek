# New-CIMCommand

## SYNOPSIS

Create a command based on a CIM class.

## SYNTAX

```yaml
New-CIMCommand [[-computername] <String>]
```

## DESCRIPTION

This command starts a graphical wizard process to help you build a new function based on a WMI/CIM class.
