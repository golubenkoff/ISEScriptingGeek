# Send-ToPrinter

## SYNOPSIS

Print the current file in the ISE.

## SYNTAX

```yaml
Send-ToPrinter [[-path] <String>]
```

## DESCRIPTION

This function is intended to be used in the ISE to print the current script file to the default printer.
