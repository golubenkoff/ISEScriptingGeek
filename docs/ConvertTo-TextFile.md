# ConvertTo-TextFile

## SYNOPSIS

Convert the file to a text file

## SYNTAX

```yaml
ConvertTo-TextFile [-Reload]
```

## DESCRIPTION

Save the current file as a txt file with an option to reload in the PowerShell ISE.
