# Edit-Snippet

## SYNOPSIS

Edit an ISE Snippet

## SYNTAX

```yaml
Edit-Snippet [[-Path] <String>]
```

## DESCRIPTION

View snippets and edit selected in the PowerShell ISE. Changes won't be effective until the next ISE session or tab.
